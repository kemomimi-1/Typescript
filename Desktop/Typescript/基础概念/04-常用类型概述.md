# TypeScript 常用类型概述

## 概述
TypeScript 的核心特性是静态类型系统。本文将概述 TypeScript 中最常用的类型，为后续深入学习打下基础。

## 基本类型

### 原始类型
```typescript
let name: string = "Alice";      // 字符串
let age: number = 25;             // 数字
let isStudent: boolean = true;    // 布尔值
let nothing: null = null;         // null
let notDefined: undefined = undefined; // undefined
```

### 特殊类型
```typescript
let anything: any = "可以是任意类型"; // 任意类型
anything = 42;
anything = true;

let unknownValue: unknown = "未知类型"; // 未知类型（比 any 更安全）
```

## 对象类型

### 对象字面量类型
```typescript
let person: { name: string; age: number } = {
  name: "Bob",
  age: 30
};
```

### 数组类型
```typescript
let numbers: number[] = [1, 2, 3];          // 数字数组
let names: Array<string> = ["A", "B", "C"]; // 泛型数组语法
```

### 元组类型
```typescript
let tuple: [string, number] = ["hello", 42]; // 固定长度的数组
```

## 函数类型

### 函数声明
```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}
```

### 函数表达式
```typescript
const add = (a: number, b: number): number => a + b;
```

### 可选参数和默认参数
```typescript
function createUser(name: string, age?: number): User {
  // age 是可选的
}
```

## 高级类型

### 联合类型
```typescript
let id: string | number = "123"; // 可以是字符串或数字
id = 456;
```

### 交叉类型
```typescript
type Named = { name: string };
type Aged = { age: number };
type Person = Named & Aged; // 同时具有 name 和 age
```

### 类型别名
```typescript
type UserID = string | number;
type Coordinate = [number, number];
```

### 接口
```typescript
interface User {
  id: number;
  name: string;
  email?: string; // 可选属性
}
```

## 字面量类型

### 字符串字面量
```typescript
type Direction = "north" | "south" | "east" | "west";
let direction: Direction = "north";
```

### 数字字面量
```typescript
type Dice = 1 | 2 | 3 | 4 | 5 | 6;
let roll: Dice = 3;
```

## 枚举类型

### 数字枚举
```typescript
enum Color {
  Red,    // 0
  Green,  // 1
  Blue    // 2
}
```

### 字符串枚举
```typescript
enum Direction {
  Up = "UP",
  Down = "DOWN"
}
```

## 类型操作

### typeof 操作符
```typescript
const point = { x: 10, y: 20 };
type Point = typeof point; // { x: number; y: number }
```

### keyof 操作符
```typescript
interface User {
  id: number;
  name: string;
}
type UserKeys = keyof User; // "id" | "name"
```

## 类型兼容性

### 结构类型系统
TypeScript 使用结构类型系统（鸭子类型），只要结构匹配就是兼容的：
```typescript
interface Named {
  name: string;
}

let x: Named;
let y = { name: "Alice", age: 25 };
x = y; // 兼容，因为 y 有 name 属性
```

## 类型推断

TypeScript 能够自动推断类型：
```typescript
let message = "hello";     // 推断为 string
let count = 42;            // 推断为 number
let values = [1, 2, 3];    // 推断为 number[]
```

## 类型层次结构

```
any
└── unknown
    ├── never
    ├── void
    ├── object
    │   ├── array
    │   ├── function
    │   └── tuple
    ├── string
    │   └── string literal
    ├── number
    │   └── number literal
    ├── boolean
    │   └── boolean literal
    └── bigint
```

## 使用建议

1. **优先使用类型推断**：让 TypeScript 自动推断简单类型
2. **明确复杂类型**：为函数参数和返回值添加类型注解
3. **避免过度使用 any**：尽量使用更具体的类型
4. **利用联合类型**：处理多种可能的情况
5. **使用接口和类型别名**：提高代码可读性和重用性
